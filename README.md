# MoneyControl Cards

Credit card comparison tool for MoneyControl. Helps users find the right card across 4 discovery flows: category browse, AI Card Genius, beat-your-card, and full catalog.

## Tech stack

- Vite + React + TypeScript
- shadcn-ui + Tailwind CSS

## Local development

```sh
npm install
npm run dev
```

App runs at http://localhost:8080
